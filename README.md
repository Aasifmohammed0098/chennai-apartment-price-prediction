# Chennai Apartment Price Prediction

## 📌 Description

This project predicts apartment prices in Chennai using Machine Learning. It analyzes features like location, size, bedrooms, and amenities to estimate accurate prices.

## 🚀 Features

* Real-time data scraping
* Multiple ML algorithms
* Price comparison (Predicted vs Actual & Government values)
* Data visualization

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib

## 📊 Output

* Top 5 expensive apartments
* Graphical insights

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells
